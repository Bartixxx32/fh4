getting started

Tips (MUST READ)

we recommend follow below tips to customize the item.

  1. do not change any css style directly, all custom style must place at assets/css/custom.css
  2. do not change assets/js/main.js directly, if you want change it directly for some reason, please mark the change, so if any update version come you can update the item easier.
  3. assets/js/variable.js is the options js file, almost all function setting true to enable, false to disable, we have comment all config part, please follow the comment to edit the file.
  4. below document wont remind above tips again, thanks :)

basic

color

open assets/js/variable.js find

var _mainColor = 'dark'; // ['light', 'dark']

replace value with light or dark

preloader

change preloader color, open assets/css/color.css find

#site-loader {
  /* site loader bg */
  background-color: rgba(255, 255, 255, 0.99);
}
.loader {
  /* loader icon */
  border-color: rgba(0, 0, 0, 0.1);
  border-left-color: rgba(0, 0, 0, 0.5);
}
.is-border #site-wrap {
  /* border */
  border-color: rgba(255, 255, 255, 0.99);
}

edit it to other color if you want :)

font

template use one google font style, open assets/css/style.css find

@import url("http://fonts.googleapis.com/css?family=Ubuntu+Condensed");

change to other font if need, and search assets/css/style.css find below font name to change

Ubuntu Condensed

if change the font style, maybe need edit some padding or line height for text/input.

title and content

to change title and content, open index.html search the text that you want to change, edit it directly.

to edit the title font size fit the screen, open assets/css/custom.css edit the font size

background

all background setting place at assets/js/variable.js, we have commend all of the options, please follow the commend to change any background version or effect.

subscribe (PHP)

open assets/php/subscribe.php find and edit

  // version, 1 = save to txt file, 2 = mailchimp
  $version = 1;

  // http://admin.mailchimp.com/account/api/
  define('MC_API_KEY', 'YOUR_MAILCHIMP_APIKEP_HERE');

  // http://admin.mailchimp.com/lists/
  define('MC_LIST_ID', 'YOUR_MAILCHIMP_ID_HERE');

  // email txt file path
  // strong recommend edit to a hard to guess file name, e.g. email_hard_to_guess.txt and rename the txt file
  // if this function not work, you may need to edit the txt file permission to 666 or above
  define('FILE_PATH', 'email.txt');
